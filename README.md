# MyEnglish — личный тренажёр английского

PWA-приложение (Progressive Web App) для самостоятельного изучения английского
по этапам **B1 → B2 → C1 → IELTS**. Работает на iPhone как обычное приложение
(иконка на экране «Домой»), офлайн, без App Store и без оплаты.

## Что внутри

Активности (каждая — с выбором уровня B1 / B2 / C1):

| Раздел | Что делает |
|---|---|
| 📖 **Чтение** | текст + вопросы на понимание |
| 🎧 **Аудирование** | тот же текст озвучивается голосом (TTS), вопросы при скрытом тексте |
| ✏️ **Грамматика** | заполнение пропусков, с подсказками и привязкой к юнитам Murphy |
| 🃏 **Слова** | флэш-карточки с интервальным повторением (SRS) |
| 🗣️ **Фонетика** | звуки, ударение + тренажёр минимальных пар на слух |
| 🔗 **Фразовые глаголы и идиомы** | карточки phrasal verbs и idioms |
| ✅ **Проверка темы** | мини-тест по грамматической теме |
| 📊 **Прогресс** | стрик, XP, статистика, путь до IELTS |
| 🎓 **Программа обучения** | дорожная карта + вся линейка учебников Cambridge «… in Use» + IELTS |

Геймификация в стиле Duolingo: **стрик** (дни подряд), **XP**, **дневная цель**.
Весь прогресс хранится на телефоне (localStorage) — ничего не уходит в интернет.

## Не один учебник, а целая программа

Приложение не привязано к одной книге. В основе — вся «лестница» Cambridge:
- **Grammar**: Essential → English (Murphy, 5th ed.) → Advanced Grammar in Use
- **Vocabulary**: Vocabulary in Use (Pre-int → Upper-int → Advanced), Collocations in Use
- **Phrasal verbs / Idioms in Use**
- **Pronunciation in Use** (Elementary → Intermediate → Advanced)
- **Listening / IELTS**: Real Listening & Speaking, Official Cambridge Guide to IELTS, Cambridge IELTS 1–18

Полный список — на экране «🎓 Программа обучения».

## Как поставить на iPhone (iPhone 14)

Приложению нужен адрес (https). Самый простой бесплатный способ — **GitHub Pages**:

1. Создать репозиторий на github.com и загрузить в него содержимое папки `english-app/`.
2. Settings → Pages → Source: ветка `main`, папка `/root` → Save.
3. Через минуту появится ссылка вида `https://<логин>.github.io/<репозиторий>/`.
4. Открыть эту ссылку **в Safari** на iPhone.
5. Нажать «Поделиться» (квадрат со стрелкой) → **«На экран „Домой“»** → «Добавить».
6. На экране появится иконка **MyEnglish** — открывается как настоящее приложение, работает офлайн.

> Альтернатива без GitHub: любой статический хостинг (Netlify drop, Cloudflare Pages, Vercel) —
> просто перетащить папку.

### Локальная проверка на компьютере
```bash
cd english-app
python3 -m http.server 8765
# открыть http://localhost:8765/index.html
```

## Как пополнять (обогащение)

Контент — обычные JS-объекты, код менять не нужно.

- **Добавить грамматику**: в `content.js` → `grammar.B2` добавь объект
  `{ id, title, murphy, topic, items: [{ q: "… ___ …", a: ["ответ"], hint }] }`.
  В `q` место пропуска — три подчёркивания `___`.
- **Добавить текст для чтения/аудирования**: `content.js` → `reading.B1` →
  `{ id, title, text, questions: [{ q, options:[…], answer: индекс }] }`.
- **Добавить слова / фразовые глаголы / идиомы**: `vocab` / `phrasal` / `idioms` →
  `{ id, en, ru, ex }`.
- **Добавить фонетику**: `phonetics` → `{ id, title, source, note, examples:[{w,ipa}], pairs:[["a","b"]] }`.
- **Добавить учебник в программу**: `curriculum.js` → массив `books` →
  `{ lvl, track, title, author, note }`.

После изменения контента подними версию кэша в `sw.js` (`myenglish-v1` → `v2`),
чтобы телефон скачал обновление.

> Проще всего: попроси Claude «добавь в программу <тему/учебник>» — он допишет контент.

## Технологии
Чистый HTML/CSS/JS без сборки и зависимостей. Озвучка — Web Speech API (встроена в Safari).
Хранение — localStorage. Офлайн — Service Worker. Установка — Web App Manifest.
